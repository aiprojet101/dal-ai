# Dal-AI — Site Agence Web

## Projet
Site vitrine pour l'agence web Dal-AI (dal-ai.com).
Creation de sites internet pour particuliers, PME et PMI.

## Stack
- Next.js 15 (App Router) + TypeScript
- Tailwind CSS v4
- Framer Motion (animations)
- Lucide React (icones)

## Structure
- `src/app/` — Pages (App Router)
- `src/components/` — Composants React (Navbar, Hero, Services, Pricing, Process, Portfolio, Testimonials, Contact, Footer)
- `src/lib/` — Utilitaires (cn)

## Design System
- Theme sombre (#050510) avec accents violet (#6c5ce7) et cyan (#00cec9)
- Glassmorphism, gradient borders, glow effects
- Polices : Geist Sans + Geist Mono

## Commandes
- `npm run dev` — Serveur de dev
- `npm run build` — Build production
- `npm run lint` — Linting

## Deploiement
Prevu sur Vercel. Domaine : dal-ai.com
